# THE DUBAI MALL | INVESTOR EXPERIENCE

A high-end, cinematic digital experience designed to showcase the unrivaled investment potential and commercial dominance of The Dubai Mall.

---

## 🚀 Setup Instructions

1. **Clone the Project**:
   ```bash
   git clone https://github.com/pratik-71/Liat-Ass.git ./
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run Locally**:
   ```bash
   npm run dev
   ```

---

## 🌊 Project Flow

The application follows a structured, high-impact narrative:

1.  **Cinematic Intro**: An immersive visual countdown presenting core metrics (300K+ daily visits) and setting the luxury tone.
2.  **Investor Dashboard**: The central hub where users can explore four distinct investment pillars through interactive "Opportunity Cards."
3.  **Deep-Dive Modules**: Each pillar provides a specialized experience with data-driven insights, brand marquee loops, and high-impact visual proof.

---

## 📦 Module Breakdown

### 💎 Retail & Leasing
*   **Focus**: Positioning brands at the heart of global retail.
*   **Key Data**: 1,200+ outlets, 98% occupancy rate, 85% luxury conversion intent.
*   **Features**: Strategic location explorer and "Choose Your Path" expansion planner.

### 🎭 Events & Platform
*   **Focus**: Scaling brand activations for a global audience.
*   **Key Data**: 100M+ annual impressions, 3.5 hours average dwell time.
*   **Features**: Infrastructure showcases and past event highlights (Luxury Showcases, Global Launches).

### 🎡 Attractions & Entertainment
*   **Focus**: Beyond retail—converting footfall into measurable commercial value.
*   **Key Data**: Global magnets like Dubai Aquarium and Reel Cinemas.
*   **Features**: Visual grid of high-peak traffic generators.

### 🍽️ Luxury & Dining
*   **Focus**: The pinnacle of commercial excellence and high-net-worth engagement.
*   **Key Data**: $1.2B+ annual luxury spend, $3,000+ average transaction value.
*   **Features**: Curated "Signature Collection" gallery.

## 🏆 Why The Dubai Mall?

*   **Unrivaled Scale**: 12 Million sq. ft. of prime real estate—the world’s largest mall.
*   **Global Reach**: Visitors from 190+ countries, ensuring a truly international brand stage.
*   **Dominant Footfall**: 100M+ annual visitors with a consistent waiting list of premium brands.

## ⚡ Tech Highlights

*   **GSAP & ScrollTrigger**: Powering the "Black & Gold" cinematic motion identity.
*   **Optimized Lazy Loading**: Custom-built `LazyImage` and `LazyVideo` components for instant rendering and high-performance asset delivery.
*   **Modular Architecture**: Built for expansion with specialized types and common UI components.
